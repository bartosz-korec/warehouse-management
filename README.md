# warehouse-management

**Warehouse Management System** is a full-stack application designed for efficient warehouse operations with intelligent employee route selection.

## Project Structure

### Backend
Located in `backend/` directory, built with:
- **Framework**: Spring Boot 3
- **Build Tool**: Maven
- **Language**: Java 21
- **Database**: Postgres 17
- **Security**: JWT

### Frontend
Located in `frontend/` directory, built with:
- **Framework**: Angular 21
- **Package Manager**: npm
- **Language**: TypeScript

## Features
- Intelligent employee route selection
- Order management
- User authentication and authorization
- Multiple warehouse grid configurations

## API Documentation

The backend provides OpenAPI/Swagger documentation:
- **Swagger UI**: `http://localhost:8080/swagger-ui/index.html`
